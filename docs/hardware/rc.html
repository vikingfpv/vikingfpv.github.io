摇杆校准流程

  1. 机械校准​：
    • 拆除弹簧组件
    • 使用0.01mm级千分尺测量中点位置

  2. 软件校准​：

    def calibrate_stick(raw_min, raw_max):
 midpoint = (raw_max + raw_min) // 2
 scale = 1000.0 / (raw_max - midpoint)
 return midpoint, scale

  3. ​动态测试​：

    • 绘制Lissajous图形验证线性度
